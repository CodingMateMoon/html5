배열 선언하기

배열메소드 활용하기

indexOf : 배열에서 요소가 위치한 인덱스 번호를 리턴

concat() : 배열을 합치는것

join : 배열을 문자열로 반환해주는 함수

sort : 내림차순이나 오름차순 정렬

배열의 데이터를 변경하는 함수

push() : 배열의 맨뒤에 값을 추가
pop() : 배열의 맨뒤값을 제거

slice(index1, index2) : 배열의 요소를 선택해서 새 배열로 만드는것
splice(startidx, deletecount, value) : 배열의 index 위치의 요소를 제거 및 추가
